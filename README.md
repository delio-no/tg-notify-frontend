# Сервис уведомлений о запланированных событиях в Telegram

Mini App для Telegram, позволяющий создавать и управлять напоминаниями о запланированных событиях с отправкой уведомлений через Telegram-бот.

## 🛠 Технологический стек

| Технология | Назначение |
|------------|------------|
| [Vue 3](https://vuejs.org/) | Фреймворк для построения пользовательского интерфейса |
| [Vite](https://vitejs.dev/) | Инструмент сборки и dev-сервер |
| [Pinia](https://pinia.vuejs.org/) | Менеджер состояния |
| [Tailwind CSS](https://tailwindcss.com/) | Утилитарный CSS-фреймворк |
| [Telegram Web App](https://core.telegram.org/bots/webapps) | SDK для интеграции с Telegram |

## 📋 Функциональность

### Типы напоминаний:

- **📅 Разовое** — указывается конкретная дата и время
- **⏰ Через...** — указывается через какое время сработает (5 минут, 1 час, 1 сутки и т.д.)
- **🔄 Еженедельное** — указывается день недели и время (каждый понедельник в 10:00)

### Общие функции:

- [x] Создание напоминаний с выбором типа
- [x] Редактирование существующих напоминаний
- [x] Удаление напоминаний
- [x] Просмотр списка всех напоминаний
- [x] Разделение на просроченные и предстоящие
- [x] Сохранение данных в localStorage
- [x] Интеграция с Telegram Mini App (темы, размер окна, данные пользователя)

## 🚀 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev
```

После запуска откройте приложение в браузере или через Telegram WebView.

## 🌐 Сборка и деплой

```bash
# Сборка для продакшена
npm run build

# Деплой на GitHub Pages
npm run deploy
```

> **Примечание:** Перед деплоем убедитесь, что в настройках репозитория GitHub Pages включен и указан правильный источник (обычно `gh-pages` branch или `root` папка).
